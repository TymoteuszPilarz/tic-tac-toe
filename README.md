# Tic-Tac-Toe

Well known, classic Tic Tac Toe game, written in C++ using SDL2 libraries. 

You can play in two modes: PvP and PvE. 

AI player in PvE mode is based on minimax algorithm (https://en.wikipedia.org/wiki/Minimax) with Alphaâ€“beta pruning optimization.

Requirements:
- C++20
- SDL2
- SDL_image
- SDL_mixer

You can download sdl libraries here: https://www.libsdl.org/projects/
